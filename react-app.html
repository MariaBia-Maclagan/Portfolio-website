Dictionary App React

HTML - CSS - JavaScript - InVision - API- GitHub - Bootstrap - Netlify - React

This interactive dictionary app was created using React. It is using two APIs - Dictionary and Pexels. Use the search bar to find the corresponding word definitions, meanings, synonyms, and images. If you are having troubles reading the phonetics there is an audio function which can help you with the pronunciation of the results. Open source code on GitHub and hosted on Netlify.

Weather App React

HTML - CSS - JavaScript - InVision - API- GitHub - Bootstrap - Netlify - React

This interactive weather app was created using React. It is using the OpenWeatherMap API giving you real-life data and forecasts for any location around the world. It will show you the current temperature, humidity, wind speed and provide you with a five-day forecast. Also uses npm animated icons. This weather app is similar to the previous project with the difference that it was created using React instead of Vanilla JS. Open source code on GitHub and hosted on Netlify.
