Weather App with JavaScript

HTML - CSS - JavaScript - Invision - API - GitHub - Bootstrap - Netlify

This interactive weather app was created using the OpenWeatherMap API giving you real-life data and forecasts for any location around the world. Find your city with the search function or simply use your current location with a single click. It will show the current temperature, humidity, wind speed and provide you with a six-day forecast. Change the location to see different comments matching the weather forecast of your inserted city. You can even choose to view the weather temperature in °C or °F whatever is more suitable for you. Open source code on GitHub and hosted on Netlify.
